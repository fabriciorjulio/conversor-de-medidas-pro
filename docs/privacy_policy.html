👋1. Introdução

Bem-vindo ao Converte Tudo ("App"). Esta Política de Privacidade descreve como coletamos, usamos e compartilhamos suas informações quando você usa nosso aplicativo móvel.

Por favor, leia esta política cuidadosamente. Se você não concordar com nossas práticas, não use este aplicativo.

Última atualização: 9 de abril de 2026

📊2. Dados Que Coletamos

2.1 Nenhum Dado Pessoal ✓

Converte Tudo NÃO coleta dados pessoais como:
  • Nome
  • Email
  • Telefone
  • Endereço
  • Informações bancárias
  • Localização

2.2 Dados Locais

Coletamos apenas dados que ficam 100% armazenados NO SEU DISPOSITIVO:

  • Histórico de conversões: últimas 10 conversões realizadas
  • Preferência de tema: se você prefere modo claro ou escuro
  • Dados de uso: quantos cálculos você fez (não é enviado)

Esses dados são armazenados localmente via SharedPreferences e nunca são enviados para servidores externos.

2.3 Permissões Solicitadas

  • Storage (Leitura/Escrita): apenas para salvar arquivos gerados (PDFs, imagens convertidas)
  • Internet: apenas para cotações ao vivo de moedas e criptomoedas

🎯3. Como Usamos Seus Dados

Usamos dados apenas para:

  1. Manter histórico de suas conversões (armazenado localmente)
  2. Permitir salvar e compartilhar resultados
  3. Melhorar a experiência do usuário (interface, performance)
  4. Fornecer cotações ao vivo (moedas, criptomoedas)

🍪4. Cookies e Rastreamento

Converte Tudo NÃO usa:
  • Cookies
  • Rastreamento de usuário
  • Analytics pessoal
  • Fingerprinting do dispositivo
  • Identificadores únicos de publicidade

📢5. Publicidade

Nosso aplicativo exibe anúncios via Google AdMob para manter a app gratuita.

Dados de publicidade:

  • Google AdMob pode coletar identificador de publicidade anônimo
  • Google AdMob segue sua própria política de privacidade
  • Você pode desativar rastreamento de publicidade em Configurações → Google → Anúncios

🔗6. Compartilhamento de Dados

Converte Tudo NÃO compartilha seus dados com terceiros, exceto:

6.1 Google Mobile Ads

  • Para exibir publicidades
  • Google segue sua própria política

6.2 Compartilhamento Manual

  • Se você clicar "Compartilhar", nós compartilhamos apenas o resultado selecionado
  • Você controla o que é compartilhado

🔒7. Segurança

Protegemos dados locais através de:

  • Armazenamento seguro: SharedPreferences com criptografia nativa do SO
  • Sem transmissão: dados nunca deixam seu dispositivo
  • Sem logins: sem senha necessária

⚖️8. Seus Direitos

Você tem o direito de:

  • Acessar: seus dados estão visíveis no app (histórico)
  • Deletar: limpe dados em Configurações do Android → Apps → Converte Tudo → Armazenamento → Limpar dados
  • Exportar: nós não temos seus dados, estão 100% em seu telefone

⏰9. Retenção de Dados

  • Histórico: mantido enquanto você não limpar (limite de 10 itens)
  • Preferências: mantidas até você resetar o app
  • Ao desinstalar: todos os dados são automaticamente removidos do dispositivo

👶10. Menores de Idade

  • Converte Tudo é adequado para todas as idades (3+)
  • Não coletamos dados de menores
  • Pais podem monitorar uso através de controles parentais do Android

📝11. Alterações Nesta Política

Podemos atualizar esta política ocasionalmente. Você será notificado de mudanças significativas através de:

  • Atualização do app na Play Store
  • Notificação in-app (quando aplicável)
