# 🌐 Como Hospedar a Política de Privacidade

Você tem um arquivo HTML pronto em: `docs/privacy_policy.html`

Agora precisa **hospedar** em um servidor para gerar uma URL pública.

---

## 3 OPÇÕES (Do Mais Fácil para o Mais Completo)

---

## **OPÇÃO 1: GitHub Pages (RECOMENDADO - Grátis e Fácil)**

### ✅ Vantagens
- Grátis para sempre
- Hospedado automaticamente do seu repositório
- URL linda: `https://fabriciorjulio.github.io/conversor-de-medidas-pro/...`
- Sem configuração necessária

### 📋 Passo-a-Passo

#### 1️⃣ Já está pronto!
Seu arquivo `docs/privacy_policy.html` já está no repositório.

#### 2️⃣ Ativar GitHub Pages
```
1. Vá para: https://github.com/fabriciorjulio/conversor-de-medidas-pro
2. Settings → Pages
3. Em "Source", selecione: Branch: main, Folder: /docs
4. Clique "Save"
```

#### 3️⃣ Aguardar 1-2 minutos
GitHub Pages vai gerar a URL automaticamente.

#### 4️⃣ Sua URL será:
```
https://fabriciorjulio.github.io/conversor-de-medidas-pro/privacy_policy.html
```

#### 5️⃣ Usar no Play Store
```
Play Console → App content → Privacy policy
Cole: https://fabriciorjulio.github.io/conversor-de-medidas-pro/privacy_policy.html
```

---

## **OPÇÃO 2: Vercel (Gratuito - Mais Profissional)**

### ✅ Vantagens
- Gratuito
- URL customizável
- Dashboard visualmente agradável
- Deploy automático

### 📋 Passo-a-Passo

#### 1️⃣ Ir para Vercel
```
https://vercel.com
```

#### 2️⃣ Fazer Login com GitHub
```
Clique "Sign Up"
Escolha "Continue with GitHub"
Autorize Vercel acessar seu repositório
```

#### 3️⃣ Importar Repositório
```
1. Clique "New Project"
2. Selecione: fabriciorjulio/conversor-de-medidas-pro
3. Clique "Import"
```

#### 4️⃣ Configurar
```
Framework: Other
Root Directory: (deixe em branco)
Clique "Deploy"
```

#### 5️⃣ Aguardar Deploy (1-2 min)
```
Status muda para "Ready"
URL gerada: https://conversor-de-medidas-pro.vercel.app
```

#### 6️⃣ Sua URL Privacidade
```
https://conversor-de-medidas-pro.vercel.app/docs/privacy_policy.html
```

#### 7️⃣ (Opcional) Customizar Domínio
```
Vercel → Settings → Domains
Adicione seu próprio domínio (se tiver)
Ex: https://convertetudo.com/privacy
```

https://netlify.com

## **OPÇÃO 4: Seu Próprio Hosting (Pago)**

Se você tiver um hosting contratado (como SiteGround, HostGator, etc):

### 📋 Passo-a-Passo

#### 1️⃣ Acessar FTP/File Manager do Hosting
```
Cpanel → File Manager
OU use FileZilla (FTP)
```

#### 2️⃣ Upload do Arquivo
```
Envie: docs/privacy_policy.html
Para: public_html/privacy_policy.html
```

#### 3️⃣ URL
```
https://seudominio.com/privacy_policy.html
```

### 1. URL Deve Estar SEMPRE Disponível
```

### 2. Mantenha Atualizado
```
Se mudar política, atualize no HTML

```

```
✅ GitHub Pages: HTTPS automático
✅ Vercel: HTTPS automático
